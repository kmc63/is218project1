Pylint

Pylint is a Python code analysis tool which can help look for programming errors, helps enforce a coding standard, it looks for code smells, and offers refactoring suggestions for your project. Pylint is very configurable since it has a very large configuration file with many options to look for errors and give you warnings in your code.

Pytest

Pytest is a testing framework that is customizable by the user and allows the user to write unit tests in the python programming language. Pytest is a great option for writing simple tests and scalable tests that interact with databases, APIs, or UIs. Although, you would usually use pytest to conduct simple tests it is also possible to create complex functional tests.
